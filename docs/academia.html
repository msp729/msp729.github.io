calculus

I think it's a pretty neat field.
It certainly contains a lot of what I consider the most interesting and powerful concepts in modern math.
I'm not aware of any notion nearly as well formalized as integrals that can solve as many problems.

abstract algebra

Quirked up set theory, busting it down structural style.
I especially like it as a motivation for defining much of math, when building it up from axioms.

set theory

This is the real "what the fuck" one, in my opinion.
y'know, the cardinality of the set of positive integers is equal to the cardinality of the set of even positive integers?

group theory

I hate all that symmetry nonsense.

linear algebra

an abomination. i despise it.
